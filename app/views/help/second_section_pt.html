2. Começar a trabalhar em Kinubi

O software de colaboração e comunicação Kinubi gira em torno dos Projetos.

O que é um Projeto?
Na platadorma utlizamos a palavra projeto para nos referir a qualquer tema onde há comunicação e troca de arquivos entre mais de uma pessoa.
Um professor pode criar um Projeto para sua aula de matemática, e convidar todos os seus alunos para colocar arquivos, abrir conversações, enviar notificações.
Para um advogado, um Projeto pode ser um cliente do seu escritório, onde necessita enviar arquivos sobre o caso, ou solicitando informações para um julgamento.
Para uma agência de publicidade, um Projeto pode ser uma camapanha publicitária de um cliente. Onde possa colocar todos os seus arquivos e solicitações que precisa para desenvolver uma campanha.
Como pode ver, um Projeto pode ser aplicado a uma infinidade de temas, em setores e áreas distintas.
Para facilitar ao usuário a classificação dos projetos que podem estar relacionados, criamos a opção Grupo de Projetos. O Grupo de Projetos é uma etiqueta agrupadora de Projetos, para que fiquem mais organizados na tabela de projetos.

De maneira muito resumida, vamos explicar os passos para começar a trabalhar por projetos. A seguir, detalhadamente como: Começar a trabalhar por projetos

  1. Crie um Grupo de projetos
  2. Crie um Projeto e atribua a um Grupo de projetos
  3. Convide usuários a participar no Projeto
  4. Comece a trabalhar: Crie conversações, Crie páginas, Carregue arquivos, Veja calendário com tarefas, Criar tarefas...

Passo 1. Crie um Grupo de projetos
Acima
Clique na opção Ambiente colaborativo/ Projetos.
Nessa opção, você poderá ver uma tabela resumida de todos os projetos em que participa.
O proprietário do Projeto é a pessoa que criou o projeto.
A tabela também inclui o papel que o seu usuário tem no projeto, é como dizer, qual a permissão você tem para trabalhar em um projeto determinado. Mais à frente, falaremos dos diferentes papéis existentes em um projeto.
Clique no botão Criar Grupo
Aqui você pode colocar um nome ao Grupo de Projetos que deseja criar.
Clique no botão Salvar.
Passo 2. Crie um Projeto e atribua à um Grupo de projetos
Acima
Clique o botão Criar Projeto
Aqui você pode colocar o nome do Projeto que deseja criar.
Atribua o Projeto à um dos Grupos de projetos que você criou.
Clique no botão Salvar.
Passo 3. Convide uisuários a participar no Projeto
Acima
Volte a selecionar a opção Ambiente colaborativo/ Projetos
Clique no projeto que deseja trabalhar.
Clique no ícone Pessoas equipe do Menu de funcionalidades do Projeto.

Introduza os e-mails ou nomes de usuários das pessoas que quer adicionar ao Projeto.
Se são vários usuários, deve incluir os e-mails ou nomes de usuário separados por vírgula.
Se a pessoa já está registrada na plataforma, aparecerá automaticamente adicionado ao Projeto.
Se a pessoa, todavia não está registrada na plataforma, receberá um convite no e-mail que você indicou, convidando-lhe a registrar-se em Kinubi.

Depois de incluir o nome de usuário, deve selecionar qual o papel que ela terá no projeto.
Por último, clique no botão Convidar.

Nota: Caso os usuários que vocè convidou tenham diferentes papéis, você terá que repetir esse passo para cada um dos papéis.
Papéis dos usuáario em um Projeto

Você pode convidar pessoas à um projeto com diferentes víveis de permissão. Faremos um resumo dos diferentes papéis que podem ser utilizados.
  • Chefe do projeto: tem acesso total à gestão de um projeto, com todas as permissões possíveis.
  • Participante: tem acesso total para trabalhar em um projeto, exceto convidar usuários ou questões específicas sobre a configuração do projeto.
  • Comentarista: tem acesso à participar em todos os elementos que foram criados no projeto.
  • Observador: pode visualizar todos os conteúdos criados no projeto.
Passo 4. Comece a trabalhar
Acima
Agora você já pode começar a trabalhar no projeto. Somente terá que ir a Ambiente colaborativo/ Projetos, clicar sobre o projeto, e começar a navegar pelos ícones do Menu de funcionalidades do projeto que necessita.
